# KopiMaster - План Развития (Roadmap)

## Фаза 1: Улучшение UX/UI (Приоритет: Высокий)

### 1.1 Онбординг для новых пользователей
- [ ] Welcome-экран с кратким туром по функциям
- [ ] Интерактивные подсказки (tooltips) при первом использовании
- [ ] Предложение создать первую цель сбережений

### 1.2 CTA (Call-to-Action) улучшения
- [ ] Плавающая кнопка "+" для быстрого добавления транзакции
- [ ] Кнопка "Поделиться" - открывает в новой вкладке страницу с достижениями
- [ ] "Пригласить друга" - генерация реферальной ссылки
- [ ] Баннер "Установить приложение" (PWA prompt)

### 1.3 PWA (Progressive Web App)
- [ ] Service Worker для офлайн-режима
- [ ] Манифест для установки на телефон
- [ ] Push-уведомления о достижении целей
- [ ] Иконки для всех платформ

## Фаза 2: Расширение функционала (Приоритет: Высокий)

### 2.1 Аналитика и отчёты
- [ ] Графики расходов по категориям (Pie chart)
- [ ] Линейный график доходов/расходов по месяцам
- [ ] Сравнение месяц к месяцу
- [ ] Экспорт отчётов в PDF
- [ ] Страница /stats (открывается в новой вкладке или модалке)

### 2.2 Бюджетирование
- [ ] Установка месячного бюджета по категориям
- [ ] Прогресс-бары использования бюджета
- [ ] Уведомления при превышении 80%, 100% бюджета
- [ ] История бюджетов

### 2.3 Повторяющиеся транзакции
- [ ] Создание регулярных платежей (аренда, подписки)
- [ ] Автоматическое добавление в указанную дату
- [ ] Напоминания о предстоящих платежах

### 2.4 Множественные счета
- [ ] Наличные, карты, сбережения
- [ ] Переводы между счетами
- [ ] Баланс по каждому счету отдельно

## Фаза 3: Социальные функции (Приоритет: Средний)

### 3.1 Достижения и геймификация
- [ ] Система достижений (бейджи)
  - "Первая транзакция"
  - "7 дней подряд"
  - "Цель достигнута"
  - "Сэкономил 1000$"
- [ ] Страница /achievements (shareable, открывается в новой вкладке)
- [ ] Уровни пользователя

### 3.2 Шеринг
- [ ] Поделиться достижением в соцсети
- [ ] Публичная страница профиля (опционально)
- [ ] Сравнение с анонимной статистикой других пользователей

## Фаза 4: Интеграции (Приоритет: Средний)

### 4.1 Импорт данных
- [ ] Импорт из CSV/Excel
- [ ] Импорт выписки банка (парсинг PDF)
- [ ] Интеграция с Monobank API (Украина)
- [ ] Plaid интеграция (США/Европа)

### 4.2 Экспорт
- [ ] Экспорт в Google Sheets
- [ ] Синхронизация с календарём (напоминания)
- [ ] Webhook для автоматизаций (Zapier, IFTTT)

## Фаза 5: Монетизация (Приоритет: Низкий)

### 5.1 Freemium модель
- [ ] Бесплатно: базовые функции, 1 счёт, 50 транзакций/месяц
- [ ] Premium: безлимит, все счета, аналитика, экспорт

### 5.2 Реклама (уже готово, нужно активировать)
- [ ] Google AdSense баннеры (после 5+ транзакций)
- [ ] Нативные спонсорские карточки
- [ ] Возможность отключить рекламу (Premium)

### 5.3 Партнёрские программы
- [ ] Рекомендации финансовых продуктов
- [ ] Cashback партнёры
- [ ] Образовательный контент от партнёров

## Фаза 6: Технические улучшения (Приоритет: Постоянно)

### 6.1 Производительность
- [ ] Виртуализация длинных списков транзакций
- [ ] Lazy loading компонентов
- [ ] Оптимизация бандла (уже частично сделано)
- [ ] Кэширование данных

### 6.2 Безопасность
- [ ] 2FA через Supabase
- [ ] Шифрование чувствительных данных
- [ ] Rate limiting
- [ ] Аудит безопасности

### 6.3 Тестирование
- [ ] Unit тесты (Vitest)
- [ ] E2E тесты (Playwright)
- [ ] Тесты доступности (a11y)

## Фаза 7: Маркетинг и рост

### 7.1 SEO и Landing
- [ ] Landing page с описанием функций
- [ ] Блог с финансовыми советами
- [ ] SEO оптимизация

### 7.2 Социальные сети
- [ ] Страница в Twitter/X

- [ ] Видео-туториалы на YouTube

## Приоритеты на ближайшее время:

1. **PWA** - чтобы пользователи могли установить как приложение
2. **Графики** - визуализация расходов
3. **Достижения** - геймификация для удержания
4. **CTA кнопки** - улучшение конверсии

## Технический стек:
- Frontend: React 18 + TypeScript + Vite
- UI: Tailwind CSS + shadcn/ui + Framer Motion
- Backend: Supabase (PostgreSQL + Auth)
- Hosting: GitHub Pages
- i18n: react-i18next (6 языков)
- Графики: Recharts (рекомендуется добавить)
